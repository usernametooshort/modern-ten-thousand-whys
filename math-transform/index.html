数学符号变换公式表

1. 符号法则

+(+a) = +a正 × 正 = 正
+(-a) = -a正 × 负 = 负
-(+a) = -a负 × 正 = 负
-(-a) = +a负 × 负 = 正

2. 括号展开

-(a + b) = -a - b
-(a - b) = -a + b
+(a + b) = a + b
+(a - b) = a - b

3. 嵌套括号 ⭐

a - (b + c) = a - b - c减号分配到每一项
a - (b - c) = a - b + c负负得正！
a + (b - c) = a + b - c加号不改变符号
a + (b + c) = a + b + c直接去括号

4. 分配律

a(b + c) = ab + ac
a(b - c) = ab - ac
-a(b + c) = -ab - ac
-a(b - c) = -ab + ac

5. 移项法则

a + b = c → a = c - b
a - b = c → a = c + b
移项要变号！

6. 合并同类项

2a + 3a = 5a
5x - 2x = 3x
系数相加减，字母不变
