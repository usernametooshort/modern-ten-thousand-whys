# 全球环流（Earth Atmospheric Circulation）

基于 Three.js 的大气三圈环流可视化，展示哈德莱、费雷尔与极地环流的热力交换与风带分布。通过粒子流线、风带标注及昼夜光照，帮助理解全球尺度的能量与动量传输。

## 功能亮点

- **三圈模型**：信风、盛行西风、极地东风以不同颜色的粒子流线呈现，展示上升/下沉与水平输送。
- **风带标注**：在 0°、30°、60°、90° 等关键纬线添加半透明环与标签，指示低压/高压区域。
- **互动控件**：侧边面板支持显示/隐藏粒子流、风带与光照，调节流速倍率，快速重置视角。
- **昼夜光照**：可一键切换风带在昼夜更替中的表现。

## 目录结构

```
earth-atmosphere/
├── index.html         # 体验入口
├── css/style.css      # 侧边 UI 与主题样式
└── js/main.js         # Three.js 场景、粒子流逻辑与 UI 交互
```

## 使用方式

1. 确保仓库根目录下的 `libs/three/` 已包含 `three.module.js` 与 `examples/jsm`。
2. 在项目根目录运行 `python3 -m http.server 4173` 或部署到 Synology Web Station。
3. 访问 `http://<host>/earth-atmosphere/` 即可体验。
